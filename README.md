# Poem
Адаптивная верстка стихотворения с элементами управления, каждый элемент отвечает 
за отображение четверостишья. Ядро адаптива - rem.

## Project Setup

## Install packages
```sh
npm install
```

### Dev-mode script (for only Development)

```sh
npm run dev
```

### Build script for deploy on Production

```sh
npm run build
```
